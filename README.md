# Democracy Engine · 民主引擎

A civilization-scale, bilingual (EN / 中文) exploration of democracy — not as elections alone, but as the **coordination technology** by which large, complex societies turn the distributed knowledge of millions of minds into legitimate, adaptive, peaceful self-government. From tribal councils and Athenian assemblies to constitutional republics, mass-media democracies, and digital, blockchain, and AI-assisted governance.

Ten interlocking systems — Origin of Democracy · Power, Legitimacy & Governance · Voting, Representation & Collective Intelligence · Information, Media & Public Opinion · Constitutions & Institutions · Democracy, Capitalism & Inequality · The Civilization Comparison Engine · Technology, Digital Democracy & AI Governance · Future Civilization Governance · The Unified Democracy Model — plus a seven-term meta-model (**Democratic Stability = Participation + Legitimacy + Information Quality + Institutional Trust + Power Distribution + Conflict Resolution + Collective Coordination Capacity**) and a recursive engine that runs the same move (many minds → legitimate collective decision) from a band of kin up to planetary coordination.

Live language toggle (no reload) and a set of hand-built canvas/SVG visualizations: a participation-network hero field, a governance-evolution ladder, a live **Stability Simulator** (participation, legitimacy, power distribution → coordination capacity, tyranny risk, fracture risk), a flagship **Voting Simulator** (the same electorate run through plurality, runoff, ranked-choice, proportional, and approval voting — watch the winner change), an **Opinion Network** with echo-chamber / polarization controls, a separation-of-powers institutional map, an inequality-vs-influence lab, a six-civilization comparison radar, a digital-governance console (blockchain / liquid / DAO / AI), a coordination-scale log chart, and a 7-axis **Democratic Stability radar**.

## Stack

Next.js 14 (App Router) · React 18 · TypeScript · Tailwind CSS · hand-built Canvas-2D / SVG visualizations · no runtime UI dependencies. Bilingual via a React-context `lang` provider (`<T v={{en,zh}}/>`). Playfair Display + Newsreader + Noto Serif SC + JetBrains Mono on a deep midnight-blue / silver / azure / teal / amber civic palette.

## About

Part of the [Psyverse](https://psyverse.fun) portfolio by [Gewenbo](https://psyverse.fun). An educational synthesis of political philosophy, social-choice theory, systems theory, economics, information theory, and governance design — it compares political systems structurally rather than ideologically, states open questions as open, and treats every simulation as an illustrative model, not a verdict.
